Quick demo

Usage

import FlashMessages from "FlashMessages"
...
< FlashMessages success={'Something went great'} error={'Something went wrong'} >
  • It prioritizes error over success in a single rendered component
  • You only need to pass it success or error as string
  • By default it empties its own state, but passing autoHide: false as a prop, it will not empty the state until the message is removed.

Options

  1. notHide: Boolean (default: false)

    2. < FlashMessages success={'Something went great'} error={''} notHide={true} >
  2. Timeout: milliseconds

    3. < FlashMessages success={'Something went great'} error={''} timeout={10000} >
  3. Style: Regular styled components values
  4. Color: Object with error or success custom text color

    5. < FlashMessages success={''} error={'Something went wrong'}
    style={{ background: 'darkred', border: '1px solid red', borderRadius: '4px' }}
    color={{ error: '#fff' }}
    >
